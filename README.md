# DSA
A comprehensive collection of fundamental Data Structures and Algorithms, featuring core data structures, advanced tree implementations, and practical file management/IO operations.

## 📂 Repository Structure

* **Basic Data Structures:** Arrays, Linked Lists, Stacks, Queues, and Hash Tables.
* **Trees:** Binary Trees, Binary Search Trees (BST), AVL Trees, and Tries.
* **File Management & I/O:** File creation, reading/writing streams, serialization, and basic file system operations.
